## これはなに
Twitterに投稿されたスプラトゥーン3のwipeout動画を表示するサービスです。

## ローカルで動かす
```
git clone git@github.com:sacckey/wipeout-ink.git

docker compose up -d
```
### ローカルページ
- トップ: http://localhost:3000
- デバッグ: http://localhost:3000/debug
- Firebase管理: http://localhost:4000
